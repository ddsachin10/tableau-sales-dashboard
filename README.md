# 📊 Sales Performance Analytics Live Dashboard – Tableau

https://public.tableau.com/app/profile/sachin.dahatonde/viz/Tableau_Sachin_WIP/Dashboard2

## 🔎 Project Summary

This project presents a **fully interactive Tableau dashboard** built to perform advanced sales performance analysis using structured business data.

The solution focuses on **trend analysis, Year-over-Year (YoY) comparison, profitability evaluation, and dimensional performance breakdown**, enabling data-driven strategic decision-making.

The dashboard is designed using best practices in **data modeling, calculated metrics, KPI engineering, and visual storytelling**.

---

## 🎯 Business Problem

Organizations require real-time visibility into:

* Revenue trends over time
* Profitability fluctuations
* Year-over-Year growth performance
* Regional and categorical contribution
* Underperforming business segments

This dashboard addresses these analytical requirements by converting raw transactional data into actionable intelligence.

## 📂 Data Engineering & Preparation

* Imported structured transactional dataset

* Validated data types and hierarchies

* Created custom date hierarchies (Year → Quarter → Month)

* Handled null values and aggregation inconsistencies

* Engineered calculated metrics including:

  * Total Sales = `SUM([Sales])`
  * Total Profit = `SUM([Profit])`
  * YoY Growth %
  * Profit Ratio = `SUM([Profit]) / SUM([Sales])`
  * Conditional KPI Indicators

* Ensured correct aggregation logic to avoid double counting
